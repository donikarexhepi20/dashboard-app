The steps to run this project are as follows:
  1.Clone the project
  2.Run npm install
  3.Run npm start

Libraries used
  Recharts / Chart.js – For interactive charts
  React-Grid-Layout – For draggable/resizable dashboard widgets

Specifications / Features
  Widget positions and layout can be saved in local storage via the Save Layout button
  Bar chart can be exported to PDF
  Pie chart opens a modal display when slices are clicked
  Authentication: User must log in to access the dashboard
  Username: admin
  Password: admin123
  State management is handled with React Hooks instead of props to improve performance and reduce parent-child complexity
